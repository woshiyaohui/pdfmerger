PDF Merger GUI
Version: 1.0
Author: Woshiyaohui
Email: x3crusift@gmail.com

==========================================
DESCRIPTION
==========================================
PDF Merger GUI is a lightweight, easy-to-use tool for combining multiple PDF files into a single document.
You can reorder, add, remove, and even drag-and-drop files before merging.

==========================================
FEATURES
==========================================
- Simple drag-and-drop interface
- Merge unlimited PDFs
- Rearrange or remove files easily
- 100% offline â€” no data uploaded
- Clean, fast, and secure

==========================================
HOW TO USE
==========================================
1. Launch PDFMerger.exe
2. Drag and drop your PDF files into the list
3. Arrange the order if needed
4. Click "Merge PDFs" and choose where to save

==========================================
REQUIREMENTS
==========================================
- Windows 10 / 11 (64-bit)
- No internet connection required
